# 机械臂 3D 可视化 - 串口功能使用文档

## 概述

本项目实现了机械臂的 3D 可视化，并支持通过串口接收数据来实时控制关节角度。

---

## 功能特性

### 1. 串口通讯
- 使用 Web Serial API 与硬件设备通讯
- 支持多种波特率（9600 - 460800）
- 实时日志显示
- 连接状态指示

### 2. 数据格式支持
系统支持以下多种数据格式：

**格式 1: 冒号分隔（推荐）**
```
J1:45 J2:90 J3:120
```

**格式 2: 等号分隔**
```
J1=45&J2=90&J3=120
```

**格式 3: JSON 格式**
```json
{"J1":45, "J2":90, "J3":120}
```

### 3. 关节绑定配置
- 支持自定义每个关节对应的串口字段名
- 默认配置：关节 1 → J1, 关节 2 → J2, ...
- 可以随时修改绑定关系

---

## 使用说明

### 第一步：连接串口

1. 在左侧面板顶部找到「串口通讯」区域
2. 选择合适的波特率（通常为 115200）
3. 点击「连接」按钮
4. 在浏览器弹出的窗口中选择你的串口设备
5. 连接成功后，状态显示为「已连接」（绿色）

### 第二步：配置关节绑定

1. 在「关节字段绑定」区域，你会看到所有关节的列表
2. 每个关节旁边有一个输入框，用于设置对应的字段名
3. 默认配置：
   - 关节 1 → J1
   - 关节 2 → J2
   - 关节 3 → J3
4. 如需修改，直接在输入框中输入新的字段名即可（不区分大小写）
5. 留空表示该关节不接收串口控制

### 第三步：发送数据

通过串口发送符合格式的数据，例如：
```
J1:45 J2:90 J3:120
```

系统会自动：
1. 解析接收到的数据
2. 根据绑定关系找到对应的关节
3. 更新关节角度（自动限制在 0-180° 范围内）
4. 在 3D 视图中实时显示

## API 说明

### SerialPortManager 类

核心串口管理类，位于 `src/utils/serialPort.ts`

**主要方法：**
- `connect(config)` - 连接串口
- `disconnect()` - 断开串口
- `onDataReceived(callback)` - 注册数据接收回调
- `onStatusChanged(callback)` - 注册状态变化回调

### useSerialData Hook

React Hook，位于 `src/utils/useSerialData.ts`

**使用示例：**
```tsx
const { status, connect, disconnect, logs } = useSerialData({
  enabled: true,
  onDataParsed: (data) => {
    console.log('收到数据:', data);
  },
});
```

### mapSerialDataToJoints 函数

将串口数据映射到关节 ID

**参数：**
- `serialData` - 串口接收的数据对象
- `bindings` - 关节绑定配置

**返回：**
- 关节 ID 到角度的映射对象

## 常见问题

### Q: 串口连接失败怎么办？
A: 
1. 确认浏览器支持 Web Serial API
2. 检查设备是否已被其他程序占用
3. 尝试重新插拔 USB 设备

### Q: 数据收到但关节不更新？
A:
1. 检查关节绑定配置是否正确
2. 确认数据格式是否符合要求
3. 查看日志面板了解详细信息

### Q: 支持哪些波特率？
A: 支持 9600, 19200, 38400, 57600, 115200, 230400, 460800
